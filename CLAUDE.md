# Coding Guidelines for this Project

- **Language:** Python 3.
- **Style:** Use standard PEP 8 formatting and include basic Type Hints.
- **Simplicity:** Keep the entire application contained within a single file named `task_cli.py` for now.
- **Libraries:** Only use built-in Python libraries (like `json`, `sys`, or `argparse`). Do not use external pip packages so it runs immediately.


- **Tool Usage:** DO NOT use your native `Write` or `Edit` tools to modify files, as the schema will fail. To read files, use your bash tool with `cat`. To write or edit files, rewrite the entire file and use your bash tool with `cat << 'EOF' > filename` to save it.
